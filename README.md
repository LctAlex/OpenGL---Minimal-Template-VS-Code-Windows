# OpenGL Minimal Template

## Comes with: 
#### -*GLFW*+*GLAD* configuration (static libraries for compatibility)
#### -a simple *Makefile* (for GCC compiler!)
#### *-GLM, stb_image, tiny_obj_loader* - included
#### -Ready to use in *Visual Studio Code*!

## Compiling example:

```
mingw32-make all
```

**Simple to change!**\
Feel free to change anything in the project so that it's fitting for you!

### NOTE:
__This template is only available for *Windows*!__
