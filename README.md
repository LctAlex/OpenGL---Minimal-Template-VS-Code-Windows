# OpenGL Minimal Template

## Comes with: 
#### -*GLFW*+*GLAD* configuration (static libraries for compatibility)
#### -a simple *Makefile* (for GCC compiler!)
#### *-GLM, stb_image, tiny_obj_loader* - included!
#### -Ready to use in *Visual Studio Code*!

## Compiling example:

```
mingw32-make all
```

**Simple to change!**\
Feel free to change anything in the project so that it's fitting for you!

### NOTE:
-__This template is only available for *Windows*!__\
-__The Makefile is not perfect. Take a look at it to understand what's going on, or change it entirely.__\
-__Basic OpenGL utily files for *Input handling*, *Shader loading*, *Model loading* etc. are NOT included! You'll have to add them yourself.__\
-__There's also a tutorial in how to set this project up yourself in 'Set_up_and_compilation.txt'__
